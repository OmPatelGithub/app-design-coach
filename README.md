# App Design Coach

Resources from a Toronto Tech Week talk on designing consumer apps that convert. 2 apps, 4.5 months, $140K revenue, $30K MRR.

## What's here

- [**app-design-coach.md**](./app-design-coach.md) — Claude Skill (and standalone prompt pack) that diagnoses onboarding, paywall, and distribution problems. Install in Claude Code or just read it.
- [**stack.md**](./stack.md) — Every tool I use to build and run apps, with links.
- [**figma-asset-maker/**](./figma-asset-maker/) — Python script that turns onboarding screen recordings into Figma-ready PNG frames.
- [**diagrams/funnel.png**](./diagrams/funnel.png) — End-to-end funnel diagram with the tool used at each stage.
- **slides/** — Talk slides (PDF). Added after the talk.

## How to use the skill

**Option 1 — open and copy.** It's a markdown file. Read it, copy any prompt out, paste into Claude or ChatGPT.

**Option 2 — install as a Claude Skill.** If you use Claude Code, drop `app-design-coach.md` into your skills directory. Claude will auto-load it whenever you ask about app design.
